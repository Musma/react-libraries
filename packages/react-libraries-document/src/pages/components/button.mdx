# Button

공통으로 사용 될 버튼 디자인입니다.  
Color는 임의로 Mcas primary 색상인 #036DB7기준(변경예정)으로 제작하며, 프로젝트 별 필요에 따라 색상을 변경합니다.

### 사용방법

```
import { Button } from '@musma/react-component'

<Button label="Something contents" />
```

### Contained Button

<Box className="not-prose flex gap-4">

  <Button size="sm" label="sm" />
  <Button size="md" label="md" />
  <Button size="lg" label="lg" />
  <Button size="full" label="Full" />
</Box>
