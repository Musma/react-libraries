## 배포 방법

1. 코드를 수정합니다.
2. pnpm changeset
3. 배포할 패키지를 선택합니다.
4. major, minor, patch 버전을 고릅니다.
5. summary에 CHANGELOG.md에 나올 내용을 적습니다.
6. git push or pull request
